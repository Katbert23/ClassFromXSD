Данная инструкция описывает механизм разбора XML-документа и создания сущностных объектов. Другими словами, на основе XML-документа формируется прокси-класс.

Инструкция:
1. У нас имеется XML-документ - Students.xml
2. Открываем документв VS
3. Создаем xml-схему. XML -> Create Schema на панели инструментов
4. Будет сформирована Xml-схема. Типы данных в схеме автоматически устанавливаются в соответствии с данными Students.xml. Если необходимо, типы данных можно изменить и, например, поставить ограничения.
5. Сохраним файлы Students.xml и Students.xsd в некоторую папку - "C:\Sample\XML"
6. Теперь нужно запустить "Visual studio command prompt"
7. В командной строке необзодимо перейти в нужный каталог и набрать следующую команду 
xsd Students.xsd /c 
(вызывается утилита xsd.exe с дополнительным параметром в краткой форме - создание классов для этой формы)
8. Таким образом будет сформирован класс для десериализации файла Students.xml
9. Теперь создаем тестовую консольку
10. Добавляем в проект созданный класс Students.cs
